===================================================================================================================

This folder's for plugin developers to place json files just in case they need such a place.
Plugin developers should put all their json files in folders named by the plugin names and place them here.
All json files here are supposed to be used by Chess plugins only.
All json files are supposed to be loaded by js files rather than the html file.
